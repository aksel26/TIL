# 분노유발자

### 코드

```c
#include <stdio.h>
#include <time.h>

int main()
{
    clock_t start, end;
    start = clock();
    double clockResult;

    int n;
    int h[101];
    int cnt = 0;
    scanf("%d", &n);

    for (int i = 0; i < n; i++)
    {
        scanf("%d", &h[i]);
    }

  // 맨뒤의 값이 최댓값으로 설정하고 맨 뒤에서 앞으로 가면서 비교한다.
    int max = h[n - 1];
  
  // i = n부터 하면 0이 포함되기 때문
    for (int i = n - 1; i >= 0; i--)
    {

      // 최댓값이 갱신될 때마다 분노유발자 수를 추가
        if (h[i] > max)
        {
            max = h[i];
            cnt++;
        }
    }
    printf("%d\n", cnt);

    end = clock(); //시간 측정 끝
    clockResult = ((double)(end - start) / CLOCKS_PER_SEC);
    printf("%f", clockResult);

    return 0;
}
```

- **배열의 맨 뒤에서 접근 !**
