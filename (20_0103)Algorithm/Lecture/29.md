# 3의 개수는 ? (Small)

### 내 코드

```c
#include <stdio.h>

int main()
{
    int n;

    scanf("%d", &n);
    int cnt3 = 0;

    for (int i = 1; i <= n; i++)
    {
        int num = i;
        while (num != 0)
        {
            int temp = num % 10;
            if (temp == 3)
            {
                cnt3++;
            }
            num /= 10;
        }
    }

    printf("%d\n", cnt3);

    return 0;
}
```

- 1부터 n까지 각 자리수를 분해해 조사하는 방법
- *1억을 넘어가면 시간초과 발생*
