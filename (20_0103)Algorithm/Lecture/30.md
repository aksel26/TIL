# 3의 개수는 ? (Large)

### 내 코드

```c
#include <stdio.h>

int main()
{
    int n, left = 1, right, cur, k = 1;

    scanf("%d", &n);
    int res = 0;
    while (left != 0)
    {
        left = n / (k * 10);
        cur = (n / k) % 10;
        right = n % k;

        if (cur < 3)
        {

            res = res + (left * k);
        }
        else if (cur > 3)
        {
            res = res + ((left + 1) * k);
        }
        else if (cur == 3)
        {
            res = res + ((left * k) + (right + 1));
        }
        k *= 10;
    }

    printf("%d\n", res);

    return 0;
}
```

- **각 자리수 마다 조사하는 방법**

  - 각 자리수마다 3보다 **큰경우, 작은경우, 같은 경우**를 나누어 생각한다.
  - **자리수를 옮겨가며** 자릿수마다의 **3이 나올 수 있는 경우**의 수를. 누적한다.
  - `int cur` : 기준이 되는 자릿수의 숫자
  - `int left` : 기준보다 왼쪽의 숫자
  - `int right` : 기준보다 오른쪽의 숫자
  - `int k` : 자릿수 (1, 10, 100, 1000, ...)
    - **cur이 3보다 큰 경우**
      - 공식 : `(left + 1) * k`
      - *00을 포함하니까 + 1 을 해준다.*
    - **cur이 3보다 작은 경우**
      - 공식 : `( left * k)`
    - **cur이 3과 같은 경우**
      - 공식 : `(left * k) + (right + 1)`
