# 버블정렬

```c
#include <stdio.h>
int a[100];
int main()
{

    int n, tmp;
    scanf("%d", &n);

    for (int i = 0; i < n; i++)
    {
        scanf("%d", &a[i]);
    }

    for (int i = 0; i < n - 1; i++)
    {
        for (int j = 0; j < n - i - 1; j++)
        {
            if (a[j] > a[j + 1])
            {
                tmp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = tmp;
            }
            else
            {
                continue;
            }
        }
    }

    for (int i = 0; i < n; i++)
    {
        printf("%d ", a[i]);
    }

    return 0;
}
```

- 가장 많은 시간복잡도를 가진다.
- 이웃하는 두 원소끼리 비교 및 swap연산을 계속 한다
- **`for (int j = 0 ; j < n - i - 1 ; j ++)`** 
  - `j<n - i -1` 조건식이 가장 핵심
  - i 가 끝까지 돌았을 경우 정렬이 되었기 때문에 다시 끝까지 갈 필요는 없기 때문.
