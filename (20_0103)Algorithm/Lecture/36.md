# 삽입정렬

```c
#include <stdio.h>
int a[100];
int main()
{

    int n, i, j;

    scanf("%d", &n);

    for (int i = 0; i < n; i++)
    {
        scanf("%d", &a[i]);
    }
    int tmp;
    for (i = 1; i < n; i++)
    {
        tmp = a[i];
        for (j = i - 1; j >= 0; j--)
        {

            if (a[j] > tmp)
                a[j + 1] = a[j];
            else
                break;
        }
        a[j + 1] = tmp;
    }

    for (int i = 0; i < n; i++)
    {
        printf("%d ", a[i]);
    }

    return 0;
}
```

- **흐름**
  1. i의 맨 처음 시작 인덱스는 1부터 시작, j는 그 앞인 i - 1
  2. 값의 이동을 위해 임시로 tmp 변수에 a[i]값을 담는다
  3. 앞으로 이동하며 값을 비교하고 크면 ( 오름차순 ) 앞의값을 뒤로 땡겨 복사한다. (`  a[j + 1] = a[j];`)
  4. 반복연산이 끝나면 복사한 값인 tmp를 치환한다.
