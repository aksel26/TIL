# LRU (Least Recently Used) 

```c
#include <stdio.h>
int c[20];
int main()
{

    int s, n, a, i, j, pos;
    scanf("%d %d", &s, &n);

    for (i = 1; i <= n; i++)
    {
        scanf("%d", &a);

        pos = -1;

        for (j = 0; j < s; j++)
        {
            if (c[j] == a)
                pos = j;
        }

        // cache miss
        if (pos == -1)
        {
            for (j = s - 1; j >= 1; j--)
                c[j] = c[j - 1];
        }
        //cache hit
        else
        {
            for (j = pos; j >= 1; j--)
                c[j] = c[j - 1];
        }
        c[j] = a;
    }

    for (i = 0; i < s; i++)
        printf("%d ", c[i]);

    return 0;
}
```

1. 복사해서 따로 담아놓고
2. 배열 값들을 땡기고
3. 담아놓은 값들을 붙여넣기

<u>작업들이 삽입정렬의 핵심인 것 같다.</u>
